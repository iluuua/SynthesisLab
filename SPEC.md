# СинтезЛаб — сайт-визитка под диагностику. Build spec v2 (LOCKED)

Канонический спек для сборки. Исполнитель — **MiMo**, ревью — CEO (Claude). Источники: academysbd.ru, ТЗ Татьяны (02.06), деки, брендбук `materials_raw/СИНТЕЗЛАБ.pdf`, ответы клиента.

## 1. Бриф (подтверждено)
- **Цель:** вести на **бесплатную диагностику** через **форму** + сайт как **визитка**.
- **Клиент №1:** собственник, HR.
- **Бренд:** СинтезЛаб. **Главный CTA везде:** «Получить диагностику» → форма.
- **Тон:** премиально, спокойно, на «вы», русский. **Директивы:** минималистично, просто, читаемо; иллюстрации из hero убрать; цвета можно менять; без перегруза.
- **Срок:** к вечеру четверга. Деплой под ключ.

## 2. Архитектура (подтверждено)
- **Одна главная страница** + **4 детальные секции услуг** с якорными ссылками из карточек. Сейчас НЕ делаем отдельные страницы (финальных текстов услуг нет).
- Контент услуг — в `src/data/services.ts` (типизировано), чтобы позже быстро собрать `/services/[slug]` из тех же данных без переписывания.
- Текст услуг придёт позже → детальные секции делаем как **слоты** с заглушкой-структурой.

## 3. Технологии и деплой
- **Astro + Tailwind**, статическая сборка, минимум JS (островки только где нужно: меню, форма).
- **Форма:** endpoint **`/api/lead`** (Astro endpoint). Поля: `имя, компания, роль, контакт, запрос`. Отправка в **Telegram + email через env** (`TELEGRAM_BOT_TOKEN`, `TELEGRAM_CHAT_ID`, `LEAD_EMAIL`...). Пока допустим **mock/dev режим** (лог + 200 OK), без реальных секретов. Honeypot + валидация.
- **Исполнитель кода:** MiMo (`mimo-v2.5-pro`, OpenGateway).
- **Деплой:** **Amvera** (наш стандарт, `deploy_targets.yaml` → `amvera_webhook_branch`, как `reach-frontend`: push в prod-ветку → сборка → nginx). v1 на домене Amvera, затем домен клиента + SSL.
- **Качество:** Lighthouse 95+, a11y, meta/OG, favicon из знака, sitemap/robots, Яндекс.Метрика (по согласованию).

**Как мы усиливаем эффективность бизнеса** (без картинок, новый визуал — чистая типо-сетка 2×2):
- *Объединяем команды* — «Создаём единую рабочую среду; повышаем вовлечённость, сохраняя ключевую экспертизу и таланты».
- *Устраняем напряжение между подразделениями* — «Выстраиваем чёткие коммуникации; убираем барьеры и конфликты между отделами».
- *Усиливаем лидеров* — «Повышаем устойчивость руководителей; помогаем уверенно управлять при высокой нагрузке».
- *Формируем единый вектор* — «Согласуем стратегию с действиями команды для работы на общий результат».

**Основные форматы работы** (4 карточки, клик → якорь на детальную секцию):
1. Командные сессии — «Синхронизируем команду в единую систему вопреки кризисам и тупикам».
2. Стратегические сессии — «Превращаем энергию хаоса в точный план действий».
3. Корпоративная медиация — «Переводим системное напряжение в ресурс для движения к целям».
4. Индивидуальный коучинг лидеров — «Точка опоры: ясность и устойчивость для точных решений».
Каждая карточка: **[Получить диагностику]** + «Подробнее →» (якорь).

**Бесплатная диагностика командной системы** (конверсионный центр): «За 50 минут вы получите: — фокусы на ближайший период — поймёте, где теряется эффективность — определитесь, куда двигаться дальше». **[Получить диагностику]** → форма.
